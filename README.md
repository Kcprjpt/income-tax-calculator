
# 📊 Advanced Income Tax Calculator & Analytics Dashboard

A modern, fast, and interactive web application built with Python to help users analyze, compare, and optimize their income tax liabilities under the **New and Old Tax Regimes (FY 2025-26 / AY 2026-27)**.

🔗 **[Live Demo Link](https://income-tax-calculator.streamlit.app/)**

## ✨ Key Features
- **Dual Regime Comparison**: Real-time side-by-side comparison of Old vs New Tax Regimes with accurate calculation of standard deductions and Section 87A rebates.
- **Interactive Visualizations**: High-performance Plotly Express charts displaying tax comparison bars and donut charts for take-home salary breakdowns.
- **Monthly Breakdowns**: Instantly displays Gross Monthly Income, Monthly Tax Deductions, and Net Take-Home Salary.
- **Smart Optimization Insights**: A fully automated English advice engine that flags missing Section 80C investments or suggests corporate NPS options based on user selections.

## 🛠️ Tech Stack
- **Framework**: Streamlit (Python Web App Interface)
- **Data Visualizations**: Plotly Express
- **Data Processing**: Pandas

## 💻 How to Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com
   cd income-tax-calculator
   ```

2. **Install requirements:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the application:**
   ```bash
   python -m streamlit run app.py
   ```
